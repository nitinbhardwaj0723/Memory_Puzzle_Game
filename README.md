# 🧠 Memory Pro Game

An advanced **Memory Puzzle Game** built using **HTML, CSS, and JavaScript** with modern UI and game mechanics.

## 🚀 Features

- 🎮 Multiple Levels (Easy → Medium → Hard)
- ⏱ Countdown Timer
- 🔥 Combo Scoring System
- 💰 Coins & Rewards System
- 💾 Data Save using LocalStorage
- 🌙 Dark Mode Toggle
- 🎨 3D Card Flip Animation
- 📱 Responsive Design

## 🧩 Game Levels

| Level | Grid Size | Time |
|------|----------|------|
| Easy | 2x2 | 20 sec |
| Medium | 4x4 | 60 sec |
| Hard | 6x6 | 120 sec |

## 🎯 How to Play

1. Click on **Start Game**
2. Flip two cards
3. Match pairs to score points
4. Complete level before time runs out
5. Unlock next level 🎉

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6 Modules)
